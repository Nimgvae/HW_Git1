*******ЗАДАНИЯ*******
Описание занятия: 1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?



Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

Как получить на локальный компьютер изменения с github.com

Какую информацию нам дают команда :
git status
git log
git remote -v 

***************Ответы**************
git status     (Команда/инструмент показывает в каком состоянии находятся файлы)
git log	       (Команда/инструмент который показывает историю коммитов(я называю это команда для вызова Хистори))
git remote -v  (Команда/инструмент чтобы посммотреть адреса для чтения и записи привязынные к репозиторию)

1.Как получить на локальный компьютер изменения с github.com 
	1. git fetch ИМЯрепозитория

2. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
	1.Прежде всего зарегистрироваться на выше указанном ресурсе(ключи,логин,пароль и т.д.)
	2.Далее установить git на свой комьютер
	3.И при помощи команды/инструмента git clone ИмяРепозитория.
	4.Вуаля, мы получаем клон(актуальный на момент скачивания) удаленного репозитория

3.Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
	1. Для начала мы запускаем git
	2. Находясь в каталоге проекта.
	3. Мы вводим git init
	4. git add *.c
	5. git add LICENSE
	6. git commit -m 'Initial project version'
